- 注册：http://localhost:8000/auth/register
- 历史：http://localhost:8000/writing/history（需登录）
- 仓库：https://github.com/Kiyoteru1225/ielts-ai-learning-platform

### 部署到 Render

1. 访问 https://dashboard.render.com
2. 点 New → Web Service → 连接 GitHub 仓库
3. 自动检测 render.yaml → 一键部署
4. 手动设置环境变量 DEEPSEEK_API_KEY
