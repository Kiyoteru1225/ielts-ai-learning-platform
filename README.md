# 雅思 AI 学习平台 (IELTS AI Learning Platform)

AI 驱动的雅思听说读写全模块学习平台，面向中国雅思考生。

## 一句话定位

为中国雅思考生提供 AI 驱动的听说读写全模块学习平台（Web 应用）。

## 功能规划

| 优先级 | 模块 | 状态 |
|--------|------|------|
| P0 | 写作批改：AI 评分 + 逐项反馈 | 规划中 |
| P0 | 用户注册/登录 | 规划中 |
| P0 | 历史记录 | 规划中 |
| P1 | 口语模拟：AI 考官对话 | 规划中 |
| P1 | 词汇模块：高频词库 + 间隔复习 | 规划中 |
| P1 | 阅读练习：AI 生成题目 | 规划中 |
| P2 | 听力练习 + 模拟考试 | 规划中 |

## 技术栈

```
后端:    FastAPI + Python 3.11+
AI:      DeepSeek API（主）+ 通义千问（备）
数据库:  SQLite → PostgreSQL
前端:    Gradio（原型）→ Jinja2 模板（MVP）
部署:    Render（原型）→ 阿里云（正式）
```

## 路线图

```
Phase 0 — 原型验证（1-2 天）
  └── Gradio 写作批改 demo + DeepSeek API

Phase 1 — 写作模块 MVP（1-2 周）
  └── FastAPI + Jinja2 + 用户系统 + Render 部署

Phase 2 — 口语 + 词汇（1-2 周）
Phase 3 — 阅读模块（1-2 周）
Phase 4 — 听力模块（1-2 周）
Phase 5 — 打磨上线（2-4 周）

总计：约 8-12 周到全功能 MVP
```

## 核心原则

1. 先跑通再优化
2. 一次只做一个模块
3. 免费优先
4. 面向中国考生
